Library simple_httpd

This library exposes the following toplevel modules:

  • Simple_httpd
  • Simple_httpd_buf Simple buffer.
  • Simple_httpd_dir Serving static content from directories
  • Simple_httpd_domain General functions related to OCaml's domain and asynchrone cooperative multithreading.
  • Simple_httpd_html HTML combinators.
  • Simple_httpd_html_
  • Simple_httpd_server HTTP server.
  • Simple_httpd_session Managment of sessions using cookies
  • Simple_httpd_stream Byte streams.
  • Simple_httpd_util
